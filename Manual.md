# Manual de Execução - Cadastro de Insumos

## Backend (.NET 8)

1. Acesse a pasta do projeto:

cd ZCK.OrdensProducao.Api


2. Configure a connection string no arquivo `appsettings.json`

3. Execute o projeto:

dotnet run


4. A API estará disponível em:

http://localhost:5188/swagger


---

## Frontend (Angular)

1. Acesse a pasta:

cd insumos-front


2. Instale dependências:

npm install


3. Execute o projeto:

ng serve

http://localhost:4200


---

## Funcionalidades

- Cadastro de insumos
- Listagem de insumos
- Exclusão de insumos
